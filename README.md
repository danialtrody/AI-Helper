## Overview

**AI Chat Project** is an **interactive chat application** powered by **Google Gemini API**, allowing users to converse with an intelligent AI assistant in **real-time**.  

The backend is built with **FastAPI**, chat history is stored in **SQLite3**, and the frontend is fully **responsive** using **Bootstrap 5**.  

Designed to be **lightweight, fast, and user-friendly**, this project provides a seamless chat experience on both desktop and mobile devices.

---

## Features

- **AI-powered chat:** Conversations are handled by Google Gemini's latest model.  
- **Multi-language support:** AI replies in the same language as the user input.  
- **Persistent chat history:** All messages are stored in SQLite3.  
- **In-memory caching:** Frequently asked messages are cached for faster responses.  
- **Responsive UI:** Works beautifully on desktop and mobile.  
- **Real-time updates:** Messages appear instantly without page reloads.  
- **Markdown normalization:** Any Markdown formatting in AI replies is stripped for clean display.

---

## Technology Stack

**Backend:**  
- FastAPI  
- SQLAlchemy (ORM)  
- SQLite3  
- Google Gemini API  

**Frontend:**  
- HTML & CSS (Bootstrap 5)  
- JavaScript (dynamic chat rendering & real-time updates)  

**Other Tools:**  
- dotenv for environment variables  
- Static file serving via FastAPI  
- In-memory caching for performance
